Portfolio

JavaScript & the Internet of Things

I participate in an open source community that is working towards making JavaScript a viable language for the future of IoT - focusing on bringing JS and the needed development tools to inexpensive microcontrollers such as the ESP8266. After participating in a React Native workshop and then receiving an ESP8266 dev board from Mozilla, I built a proof-of-concept that lets me remotely communicate with my IoT device using a React Native app through a Node/Express server hosted on Heroku.

GitHub Repo

React JS & Math

Looking for an opportunity to build something new with React, I decided to make a stretchy calculator that would conform to whatever shape it's being displayed in. Thus my Big Ass Calculator was born! It's built with React and Sass relying heavily on CSS3's vh and vw units. To overcome problems with floating point calculations, I put together a micro-library called sin-math based off of a fork from sinful.js.
